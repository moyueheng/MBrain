# 命令面板

[[tolaria-principles]] 的核心理念之一是键盘优先。几乎所有操作都有快捷键。

你可以通过 Mac 菜单栏和按钮上的 tooltip 来发现快捷键。

许多操作还提供了备用方式或快捷入口，可以通过命令面板来完成，使用 `cmd+K` 即可打开。

你几乎可以用它完成所有操作，包括导航到不同部分、创建特定类型的新笔记，或执行日常操作。
