# Git 与 AutoGit

Tolaria vault 提供一流的 Git 支持，默认情况下你可以对其进行精细控制。

你可以从 [[tolaria-bottom-bar]] 查看当前的更改和 Git 历史，无需离开应用即可直接创建新 commit 并推送。

如果你不想手动管理这些，Tolaria 还提供了一个可选的 **AutoGit 模式**——它会在一段时间后、应用空闲时、你切换到其他应用窗口时，以及基于各种启发式规则，自动创建 commit 并推送更改。

![CleanShot 2026-04-17 at 17.44.08@2x.png](attachments/1776440668823-CleanShot_2026-04-17_at_17.44.08_2x.png)

你可以在设置（`cmd+,`）中开启/关闭 AutoGit，并调整一些参数。
